About Document Verification

This project is designed to help verify Voter ID documents by checking for essential fields such as:

  • Clearly visible name and photograph
  • Presence of a valid Voter ID number
  • Proof of Date of Birth

Users can upload their Voter ID file and mark the criteria they meet. The system checks whether at least 3 essential conditions are satisfied and displays whether the form is correct or incomplete.

Technologies Used:

  • Frontend: HTML, CSS, Bootstrap 5, JavaScript
  • Particle Animation: Particles.js
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • File Upload: Multer Middleware
