Help & Support

How to Upload a Document?

Click on the "Upload" or "Submit Document" button on the home page. Choose your Voter ID file and mark the required checkboxes (Name, Photo, DOB, etc.). Click on "Submit" to check if your document meets the required criteria.

What Are the Required Fields?

  • Clearly visible Name
  • Photograph
  • Valid Voter ID Number
  • Date of Birth
  • Signature (if required)

At least 3 of the above must be present for a document to be marked as Correct.

Getting Errors While Uploading?

If you're facing upload issues, ensure:

  • The file is in JPG, PNG, or PDF format
  • The file size is less than 5MB
  • You have marked the required checkboxes
